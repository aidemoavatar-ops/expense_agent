# Expense Agent

An AI-powered expense management agent built with Python.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

## Development

Run tests:
```bash
pytest
```

Run linter:
```bash
ruff check .
```
